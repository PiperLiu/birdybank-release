# BirdyBank Release
# BirdyBank 发布

This repository hosts installers and release notes for BirdyBank (a desktop bird photo manager).

这个仓库用于存放 BirdyBank（桌面端鸟类照片管理工具）的安装包与发布说明。

BirdyBank is **desktop-only** (macOS / Windows) and focuses on local-first organization.

BirdyBank **仅桌面端**（macOS / Windows），主打本地优先的照片整理体验。

## Download
## 下载

Get the latest version from GitHub Releases:

从 GitHub Releases 获取最新版：

- GitHub Releases: https://github.com/PiperLiu/birdybank-release/releases

If you are in Mainland China and downloads are slow, use the Gitee mirror Releases:

如果你在中国大陆下载较慢，建议使用 Gitee 镜像的 Releases：

- Gitee mirror: https://gitee.com/piperliu/birdybank-release

## Install
## 安装

Choose the asset that matches your OS and CPU, then download and run it.

选择与你的系统与 CPU 匹配的安装包，下载后直接运行即可。

macOS typically uses `.dmg` (or `.app` in a zip).

macOS 通常使用 `.dmg`（或 zip 内的 `.app`）。

Windows typically uses `.exe` installer (or `.msix` if provided).

Windows 通常使用 `.exe` 安装包（若提供也可能有 `.msix`）。

## Notes for first run
## 首次运行提示

On macOS, Gatekeeper may block unsigned apps; right-click → Open can usually proceed.

在 macOS 上，Gatekeeper 可能会拦截未签名应用；通常可右键 → 打开继续运行。

On Windows, SmartScreen may warn for new apps; you can choose “More info” → “Run anyway”.

在 Windows 上，SmartScreen 可能会对新应用弹窗；可选择“更多信息”→“仍要运行”。
